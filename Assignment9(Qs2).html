key properties in the Flexbox layout model

  1. justify-content

    Two axis is there - one is main or x-axis (used horizontally) and the other is cross-axis or y-axisn(used vertically). The justify-content is a flex property which is used to align the items along the main-axis of a flex container.
    justify-content can have different values : justify-content: flex-start|flex-end|center|space-between|space-around|space-evenly|initial|inherit;

  2. align-items

    The align-items is a flex property which is used to align the items along the cross-axis of a flex container (which is vertical by default).
    align-items can have different values : align-items: normal|stretch|positional alignment|flex-start|flex-end|baseline|initial|inherit;

    The gap property defines the size of the gap between the rows and between the columns in flexbox

  4. flex-direction

    The flex-direction property specifies the direction of the flex items.
    flex-direction can have different values :flex-direction: row|row-reverse|column|column-reverse|initial|inherit;

    The flex-wrap property specifies whether the flex items should wrap or not.
